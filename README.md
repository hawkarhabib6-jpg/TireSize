# قەبارەی تایە (TireSize)

ئەپی ئەندرۆید بۆ دۆزینەوەی قەبارەی تایەی ئۆتۆمبیلەکانی بازاڕی عێراق و کوردستان بەپێی مۆدێل و ساڵ.

## تایبەتمەندییەکان
- گەڕان بە ئینگلیزی، کوردی و عەرەبی (بۆ نموونە: `Altima 2017` یان `ئالتیما` یان `التيما`)
- گەڕان بە ساڵ: تەنها `2017` بنووسە، هەموو ئەو مۆدێلانە دەردەکەون کە ئەو ساڵە دەگرنەوە
- بێ ئینتەرنێت کار دەکات — هەموو داتاکە لەناو ئەپەکەیە
- زیادکردنی ئۆتۆمبیلی نوێ زۆر ئاسانە: تەنها فایلی `app/src/main/assets/cars.json` دەستکاری بکە

## چۆنیەتی دروستکردنی APK بەبێ Android Studio (بە GitHub)
1. هەژمارێک لە github.com دروست بکە
2. ڕیپۆزیتۆرییەکی نوێ دروست بکە (بۆ نموونە `TireSize`)
3. هەموو فایلەکانی ئەم پرۆژەیە باربکە (upload) بۆ ڕیپۆزیتۆرییەکە
4. بڕۆ بۆ بەشی **Actions** — خۆی دەستپێدەکات و APK دروست دەکات (نزیکەی ٥-١٠ خولەک)
5. لە کۆتایی، لە بەشی **Artifacts** فایلی `TireSize-debug-apk` دابگرە
6. فایلە zip ەکە بکەرەوە، `app-debug.apk` بنێرە بۆ مۆبایلەکەت و دایبمەزرێنە

## چۆنیەتی زیادکردنی ئۆتۆمبیل
لە فایلی `cars.json` دێڕێکی نوێ زیاد بکە بەم شێوەیە:

```json
{"brand":"Toyota","model":"Camry","yearFrom":2018,"yearTo":2024,"sizes":["215/55R17"],"aliases":["کامری","كامري"]}
```

پاشان دووبارە فایلەکان باربکە بۆ GitHub — خۆی APK ی نوێ دروست دەکات.

## تێبینی
قەبارەی تایە بەپێی تریمی ئۆتۆمبیلەکە دەگۆڕێت. هەمیشە پلاکی ناو دەرگای شۆفێر بپشکنە بۆ دڵنیابوونی تەواو.

## گەڕانی ئۆنلاین (AI)
ئەگەر ئۆتۆمبیلێک لە داتابەیسەکەدا نەبوو، دوگمەی «🌐 گەڕانی ئۆنلاین» دەردەکەوێت و لە زیرەکی دەستکردی Claude دەپرسێت.

بۆ کارکردنی ئەم بەشە:
1. بڕۆ بۆ **console.anthropic.com** و هەژمارێک دروست بکە
2. لە بەشی **API Keys** کلیلێک دروست بکە (بە `sk-ant-` دەست پێ دەکات)
3. یەکەم جار کە دوگمەی گەڕانی ئۆنلاین دادەگریت، ئەپەکە داوای کلیلەکە دەکات — دایبنێ و هەڵیبگرە
4. کلیلەکە تەنها لەسەر مۆبایلەکەت هەڵدەگیرێت. بۆ گۆڕینی، پەنجەت درێژ ڕابگرە لەسەر دوگمەی گەڕانی ئۆنلاین

تێبینی: بەکارهێنانی API پارەی پێدەچێت بەپێی ژمارەی پرسیارەکان (هەر گەڕانێک بڕێکی زۆر کەمە). ئەنجامی AI هەمیشە لەگەڵ پلاکی ناو دەرگای شۆفێر بەراورد بکە.
